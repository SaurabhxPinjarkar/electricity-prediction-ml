# Data Folder

This folder contains the datasets required for the Electricity Consumption Prediction model.

## Files

- **energy_dataset.csv**: Historical electricity consumption data with timestamps
- **weather_features.csv**: Weather data (temperature, humidity, wind, etc.) with timestamps

## For Streamlit Cloud Deployment

These CSV files are required for the app to work on Streamlit Cloud. Make sure these files are uploaded to your GitHub repository.

## File Size

If your CSV files are larger than 100MB, you have two options:

1. **Sample the data**: Reduce the number of rows
2. **Use Git LFS**: Install Git Large File Storage
3. **External hosting**: Upload to cloud storage and download in code
